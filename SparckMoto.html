Project Overview

For my senior Capstone project at Oregon State University, my team worked on an E-Commerce website for motorcycle parts. Our team used ASP.NET MVC Entity Framework. We operated using an Agile Methodolgy.

My Main areas of focus on this project were:

  • Building functionality for site administrators to add new products to the database.
  • Implementing features allowing store customers to add, remove, update, and view items in their shopping cart.
  • Build checkout functionality using the Paypal API.
  • Viewing a dasboard for site admins to view all the store's products, sortable by name, part number, price, etc.
